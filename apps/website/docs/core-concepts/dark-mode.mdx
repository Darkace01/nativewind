# Dark Mode

NativeWind includes a `dark` variant that lets you style your site differently when dark mode is enabled

:::caution
Expo apps must set `userInterfaceStyle: 'automatic'` and install `expo-system-ui`. [Please see the Expo docs for more information.](https://docs.expo.dev/guides/color-schemes/)
:::

## Toggling dark mode manually

To enable manual control of Dark Mode, you first need to set `darkMode: 'class'` in your `tailwind.config.js`

```diff
// tailwind.config.js

module.exports = {
+ darkMode: 'class',
  // ...
}
```

You can then use the [`useColorScheme()`](../../api/use-color-scheme) hook to manually control the color scheme.

```SnackPlayer
---

---
import { Pressable, Text } from "react-native"
import { styled, useColorScheme } from "nativewind";

const StyledPressable = styled(Pressable)
const StyledText = styled(Text)

function App() {
  const { colorScheme, toggleColorScheme } = useColorScheme();

  return (
    <StyledPressable
      onPress={toggleColorScheme}
      className="flex-1 items-center justify-center dark:bg-slate-800"
    >
      <StyledText
        selectable={false}
        className="dark:text-white"
```

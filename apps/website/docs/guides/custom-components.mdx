# Custom Components

NativeWind's strength really shines when writing custom components. To merge styles, you simply combine the className strings together. Additionaly, NativeWind handles merging with inline styles!

```tsx
function MyComponent({ classes, style }) {
  const classes = `text-black`;
  return <Text className={`font-bold ${classes}`} style={style} />;
}
```

This pattern is very useful for creating complex styles based upon attributes

```tsx
const variantStyles = {
  primary: "bg-blue-500 text-white",
  secondary: "bg-white-500 text-black",
};

function MyComponent({ variant }) {
  return (
    <Text className={`font-bold rounded ${variantStyles[variant]}`} />
  );
}
```

## Reducing boilerplate

NativeWind helps reduce the boilerplate of your components with the [`styled()`](../api/styled) helper. We can rewrite the component from before as

```tsx
const MyComponent = styled(Text, "font-bold rounded", {
  variants: {
    intent: {
      primary: "bg-blue-500 text-white",
      secondary: "bg-white-500 text-black",
    }
  }
})
```

## Manually writing variants

The variant functionality of `styled()` is also available anytime though the `variants()` helper.

```tsx
import { variants } from "nativewind"

const myComponentVariants = variants({
  intent: {
    primary: "bg-blue-500 text-white",
    secondary: "bg-white-500 text-black",
  }
})

function MyComponent(props) {
  return (
    <Text className={`font-bold rounded ${myComponentVariants(props)}`} />
  );
}
```

Or you can choose a 3rd Party solution


```tsx
import clsx from "clsx"

function MyComponent({ rounded, className }) {
  const className = clsx(
    "font-bold rounded",
    variant === "primary" && "bg-blue-500 text-white",
    variant === "secondary" && "bg-blue-500 text-white",
    className
  );
  return <Text className={className} />;
}
```

## Gotcha's

### Passing styles to components

When passing styles between components, they are compiled on the parent and passed as a `style` prop to the child.

```tsx
function MyComponent({ style }) {
  // You need to use style, className is not passed down!
  return <Text style={style} />;
}

<MyComponent className="font-bold" />
```

If you want to pass the className as a string use a different prop.

```tsx
function A({ style }) {
  return <B textClassName="font-bold" />;
}

function B({ textClassName }) {
  return <Text className={textClassName} />;
}
```
